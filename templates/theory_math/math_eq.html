Уравнения и неравенства

Задача №1
Линейное уравнение

Решите уравнение: \(-x-4+5\left(x+3\right)=5\left(-1-x\right)-2\).

Показать решение

Решение:

Уравнение линейное. Раскрываем скобки, приводим подобные слагаемые, все «иксы» переносим в левую часть равенства, всё без «иксов» – вправо:

\(-x-4+5x+15 = -5-5x-2\)

\(4x+11 = -5x-7\)

\(4x+5x = -11-7\)

\(9x = -18 \Rightarrow x = -2\)

Ответ: \(-2\)

Задача №2
Квадратное уравнение

Решите уравнение: \(\displaystyle\frac{5}{4}x^2+7x+9=0\). Если корней несколько, запишите их в ответ без пробелов в порядке возрастания.

Показать решение

Решение:

Уравнение квадратное: \(a=\frac{5}{4},\; b=7,\; c=9\).

Вычисляем дискриминант: \(D = b^2 - 4ac = 7^2 - 4\cdot\frac{5}{4}\cdot9 = 49 - 45 = 4\).

Корни: \(x_{1,2} = \frac{-b \pm \sqrt{D}}{2a} = \frac{-7 \pm 2}{2\cdot\frac{5}{4}} = \frac{-7 \pm 2}{\frac{5}{2}} = (-7 \pm 2)\cdot\frac{2}{5}\)

\(x_1 = (-7-2)\cdot\frac{2}{5} = -9\cdot0,4 = -3,6\)

\(x_2 = (-7+2)\cdot\frac{2}{5} = -5\cdot0,4 = -2\)

Ответ: \(-3,6-2\)

Задача №3
Произведение = 0

Решите уравнение: \((-4x-3)(x-3)=0\). Если уравнение имеет более одного корня, в ответ запишите меньший из корней.

Показать решение

Решение:

Произведение равно нулю, когда хотя бы один из множителей равен нулю:

\(-4x-3 = 0 \Rightarrow x = -\frac{3}{4} = -0,75\)

или \(x-3 = 0 \Rightarrow x = 3\)

Меньший из корней: \(-0,75\)

Ответ: \(-0,75\)

Задача №4
Система уравнений

Решите систему уравнений: \(\left\{ \begin{array}{c} 3x+2y=8 \\ 4x-y=7 \end{array} \right.\) В ответе запишите значение \(x+y\).

Показать решение

Решение:

Используем метод подстановки: из второго уравнения выражаем \(y = 4x-7\) и подставляем в первое:

\(3x + 2(4x-7) = 8\)

\(3x + 8x - 14 = 8\)

\(11x = 22 \Rightarrow x = 2\)

\(y = 4\cdot2 - 7 = 8 - 7 = 1\)

\(x + y = 2 + 1 = 3\)

Ответ: \(3\)

Задача №5
Графики функций

На рисунке изображены графики функций \(y=4-x^2\) и \(y=-2-x\). Вычислите ординату точки \(B\).

Рис. 1 — Графики функций \(y=4-x^2\) и \(y=-2-x\)
Показать решение

Решение:

Координаты точек пересечения находим из системы уравнений:

\(\left\{ \begin{array}{c} y = 4-x^2 \\ y = -2-x \end{array} \right. \Rightarrow -2-x = 4-x^2 \Rightarrow x^2 - x - 6 = 0\)

\(D = 1 + 24 = 25\), \(x_{1,2} = \frac{1 \pm 5}{2}\)

\(x_1 = -2\) (точка А), \(x_2 = 3\) — абсцисса точки \(B\)

Ордината точки \(B\): \(y = -2 - 3 = -5\)

Ответ: \(-5\)

Задача №6
Система неравенств

Найдите наибольшее значение \(x\), удовлетворяющее системе неравенств: \(\left\{ \begin{array}{c} 8x+16 \le 0 \\ 2-2x < 13 \end{array} \right.\)

Показать решение

Решение:

Решаем каждое неравенство отдельно:

\(8x+16 \le 0 \Rightarrow 8x \le -16 \Rightarrow x \le -2\)

\(2-2x < 13 \Rightarrow -2x < 11 \Rightarrow x > -5,5\)

Пересечение решений: \((-5,5; -2]\)

Наибольшее значение \(x = -2\)

Ответ: \(-2\)
