Функции и графики

Линейная функция, обратная пропорциональность, квадратичная функция, степенная функция. Чтение графиков, свойства функций, определение коэффициентов.

4 типа функций Все свойства и графики
Основные понятия
Теория
Что такое функция?

Функция — это зависимость переменной \(y\) от переменной \(x\), при которой каждому значению \(x\) соответствует единственное значение \(y\).

\(x\) — аргумент (независимая переменная)

\(y\) — зависимая переменная (значение функции)

Важные определения

Область определения — все значения, которые может принимать \(x\).

Область значений — все значения, которые принимает \(y\).

Нули функции — значения \(x\), при которых \(y = 0\).

График функции — множество всех точек \((x; y)\) на координатной плоскости.

1
Линейная функция
Общий вид

\(y = kx + b\)

где \(x\) — аргумент, \(k, b \in \mathbb{R}\). График — прямая линия.

\(k\) — угловой коэффициент. \(b\) — свободный член (точка пересечения с осью \(y\)).

Нуль функции

\(x = -\frac{b}{k}\) (при \(k \neq 0\))

Знаки функции

При \(k > 0\): \(y > 0\) при \(x > -\frac{b}{k}\), \(y < 0\) при \(x < -\frac{b}{k}\)

При \(k < 0\): \(y > 0\) при \(x < -\frac{b}{k}\), \(y < 0\) при \(x > -\frac{b}{k}\)

\(k > 0\)

Функция возрастает на \(\mathbb{R}\)

Прямая идёт вверх

\(k < 0\)

Функция убывает на \(\mathbb{R}\)

Прямая идёт вниз

\(b\)

Точка пересечения с осью \(y\): \((0; b)\)

Рис. 1 — График линейной функции y = kx + b
2
Обратная пропорциональность
Общий вид

\(y = \frac{k}{x}\)

где \(x\) — аргумент, \(k \in \mathbb{R}\), \(k \neq 0\). График — гипербола.

Область определения

\(x \neq 0\) (все числа, кроме нуля)

Область значений

\(y \neq 0\) (все числа, кроме нуля)

\(k > 0\)

Ветви гиперболы расположены в I и III четвертях

Функция убывает на \((-\infty; 0)\) и \((0; +\infty)\)

\(k < 0\)

Ветви гиперболы расположены в II и IV четвертях

Функция возрастает на \((-\infty; 0)\) и \((0; +\infty)\)

Рис. 2 — График обратной зависимости
3
Квадратичная функция
Общий вид

\(y = ax^2 + bx + c\)

где \(x\) — аргумент, \(a, b, c \in \mathbb{R}\), \(a \neq 0\). График — парабола.

Вершина параболы

\(x_0 = -\frac{b}{2a}\)

\(y_0 = -\frac{b^2 - 4ac}{4a}\)

Ось симметрии

Прямая \(x = -\frac{b}{2a}\)

\(a > 0\)

Ветви параболы направлены вверх

\(y_{\min} = y_0\)

Убывает на \((-\infty; x_0]\), возрастает на \([x_0; +\infty)\)

\(a < 0\)

Ветви параболы направлены вниз

\(y_{\max} = y_0\)

Возрастает на \((-\infty; x_0]\), убывает на \([x_0; +\infty)\)

Частный случай: \(y = ax^2\)

График проходит через начало координат. Симметричен относительно оси \(y\).

При \(a > 0\): \(y_{\min} = 0\) при \(x = 0\)

При \(a < 0\): \(y_{\max} = 0\) при \(x = 0\)

Преобразования графика \(y = ax^2\)

\(y = ax^2 + m\) — сдвиг вдоль оси \(y\) на \(m\) единиц (вверх при \(m>0\), вниз при \(m<0\))

\(y = a(x - m)^2\) — сдвиг вдоль оси \(x\) на \(m\) единиц (вправо при \(m>0\), влево при \(m<0\))

Рис. 3 — График квадратичной функции
4
Степенная функция
Общий вид

\(y = x^n\)

где \(x\) — аргумент, \(n \in \mathbb{N}\) (натуральное число)

Чётная функция

Определение: \(f(-x) = f(x)\)

График симметричен относительно оси \(y\)

При чётном \(n = 2k\):
  • ✓ \(y > 0\) при \(x \neq 0\)
  • ✓ Функция чётная
  • ✓ Убывает на \((-\infty; 0]\), возрастает на \([0; +\infty)\)
  • ✓ Область значений: \([0; +\infty)\)
Нечётная функция

Определение: \(f(-x) = -f(x)\)

График симметричен относительно начала координат

При нечётном \(n = 2k-1\):
  • ✓ \(y > 0\) при \(x > 0\), \(y < 0\) при \(x < 0\)
  • ✓ Функция нечётная
  • ✓ Возрастает на \(\mathbb{R}\)
  • ✓ Область значений: \(\mathbb{R}\)
Рис. 4 — График степенной функции \(y = x^n\): при чётном \(n\) — парабола, при нечётном \(n\) — кубическая парабола
5
Монотонность
Возрастающая функция

Для любых \(x_1 < x_2\) выполняется \(f(x_1) < f(x_2)\)

Большему аргументу — большее значение функции

Убывающая функция

Для любых \(x_1 < x_2\) выполняется \(f(x_1) > f(x_2)\)

Большему аргументу — меньшее значение функции
