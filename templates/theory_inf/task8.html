Задание №8: Теория множеств и круги Эйлера

1. Основные операции с множествами

Пересечение (И, &)

A & B = A ∩ B

Элементы, которые одновременно присутствуют и в A, и в B.

Объединение (ИЛИ, |)

A | B = A ∪ B

Все элементы из обоих множеств без повторений.

2. Основные формулы

Формула двух множеств:

A + B = A|B + A&B

Формула трёх множеств:

A + B + C + A&B&C = A|B|C + A&B + A&C + B&C

Формула трёх множеств с нулём:

Если A&C = 0, то: A + B + C = A|B|C + A&B + B&C

3. Пример решения задания №8

Условие:
Поисковые запросы:
• Перри & Финес = 793
• Финес & (Перри | Изабелла) = 811
• Перри & Изабелла & Финес = 454

Найти: Финес & Изабелла = ?

Решение:
1. Так как Финес есть во всех запросах, его можно вычеркнуть
2. Задача сводится к двум множествам: Перри и Изабелла
3. Перри = a + b = 793
4. Перри | Изабелла = a + b + c = 811
5. Перри & Изабелла = b = 454
6. Изабелла = b + c
7. Находим a: a = (a+b) - b = 793 - 454 = 339
8. Находим b+c: b+c = (a+b+c) - a = 811 - 339 = 472

Ответ: 472

  3. Записать уравнения для каждого запроса
