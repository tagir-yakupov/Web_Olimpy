Задание №7: Адресация в сети Интернет

1. URL-адрес (адрес файла в сети)

протокол://сервер/каталог/файл.расширение
протокол
://
сервер
/
каталог
/
файл.расширение
Пример:
http://videoday.com/video/game.mp4

• протокол: http
• сервер: videoday.com
• каталог: video
• файл: game.mp4

2. IP-адрес

XXX.XXX.XXX.XXX

Правила:

  • IP-адрес состоит из четырёх чисел, разделённых точками
  • Каждое число не должно превышать 255
  • Адрес не должен начинаться с точки
  • Не должно быть нескольких точек подряд
Пример сборки IP-адреса из осколков:
Дано: A=75, B=15, C=3.14, D=.96
Правильный порядок: 153.14.75.96
Ответ: BCAD

3. Электронная почта (Email)

имя_ящика@имя_сервера.домен
Пример:
• имя ящика: andrew2005
• сервер: stud.ru
• Полный адрес: andrew2005@stud.ru

4. Пример решения

Условие:
Михаил создал документ, сохранил его на компьютере с именем http и расширением xlsx. Затем он отправил его на сервер "shake.com", доступ к которому выполняется по протоколу rtf.

Решение:
Применяем правило: протокол://сервер/каталог/файл.расширение
Получаем: rtf://shake.com/http.xlsx

Ответ: последовательность букв, кодирующих адрес
